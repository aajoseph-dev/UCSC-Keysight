ABORt Subsystem

ABORt [(@<chanlist>)]

The command clears any pending delayed trigger and returns the trigger system to idle. If INIT:CONT is enabled, ABORt clears any pending delayed trigger and re-initiate them. The INIT:CONT state will remain unchanged. This command also resets the WTG bit in the Questionable Instrument Summary status register.

Parameter Typical return

(@<chanlist>)

 (none)
Aborts the triggered action of channel 1: ABOR (@1)
