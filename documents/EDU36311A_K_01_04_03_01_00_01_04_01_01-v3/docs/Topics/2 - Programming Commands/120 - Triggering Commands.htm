Triggering Commands

The instrument's triggering system allows you to change voltage and current output when a trigger is received. The typical process is:

  1. Select one or more outputs with the INSTrument:SELect command.
  2. Configure the triggered output levels by using CURRent:TRIGgered and VOLTage:TRIGgered.
  3. Configure the voltage and current mode to Step by using VOLTage:MODE and CURRent:MODE.
  4. Specify the trigger source, either BUS or IMMediate.
  5. If you are using the BUS trigger source, you may choose to set a time delay between the detection of the trigger and the start of any corresponding output change.
  6. Send an INITiate[:IMMediate]command. If the IMMediate source is selected, the selected output is set to the triggered level immediately. If the BUS trigger source is selected, the output is set to the triggered level after the instrument receives the *TRG command.
