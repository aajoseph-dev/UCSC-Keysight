Based on the context provided, I will generate a class in Python that will serve as a plugin for the EDU36311A power supply device, allowing you to control the display using SCPI commands. The class will include functions to enable and disable the display as well as to query the current status of the display.

Here is an example of how you might implement such a class:

```python
import visa

class EDU36311APowerSupply:
    def __init__(self, resource_name):
        # Initialize VISA resource manager and open a connection to the instrument
        self.rm = visa.ResourceManager()
        self.instrument = self.rm.open_resource(resource_name)
    
    def enable_display(self, enable=True):
        # Turns the display on (True) or off (False)
        value = 'ON' if enable else 'OFF'
        self.instrument.write(':DISPlay:ENABle {}'.format(value))
    
    def query_display_enable(self):
        # Queries the current display enable status
        response = self.instrument.query(':DISPlay:ENABle?')
        return True if response.strip() == '1' else False

    def close(self):
        # Close the connection to the instrument
        self.instrument.close()

# Usage example
if __name__ == "__main__":
    # Replace 'USB0::0x2A8D::0x1202::MY12345678::INSTR' with the actual VISA address of your device
    power_supply = EDU36311APowerSupply('USB0::0x2A8D::0x1202::MY12345678::INSTR')

    # Enable the display
    power_supply.enable_display(True)
    print("Display enabled status:", power_supply.query_display_enable())

    # Disable the display
    power_supply.enable_display(False)
    print("Display enabled status:", power_supply.query_display_enable())

    # Always close the connection when done
    power_supply.close()
```

This code assumes you have the PyVISA package installed, which is a Python library for interfacing with instruments using the VISA standard. You can install it using pip:


You'll also need a VISA implementation (such as NI-VISA) installed on your machine for PyVISA to communicate with the instrument. Make sure to replace the `resource_name` parameter with the actual VISA address of your EDU36311A power supply.
