To create a Python class for controlling the EDU36311A power supply using SCPI commands, we can use the `pyvisa` library to communicate with the device over a USB interface. Below is an example class that includes methods for controlling the display of the EDU36311A:

```python
import pyvisa

class EDU36311A:
    def __init__(self, resource_name):
        # Initialize VISA resource manager
        self.rm = pyvisa.ResourceManager()
        # Open the connection to the device
        self.instr = self.rm.open_resource(resource_name)
    
    def close(self):
        # Close the connection to the device
        self.instr.close()
    
    def enable_display(self, enable):
        # Enable or disable the display
        if enable:
            self.instr.write(':DISPlay:ENABle ON')
        else:
            self.instr.write(':DISPlay:ENABle OFF')
    
    def query_display_enabled(self):
        # Query if the display is enabled
        return self.instr.query(':DISPlay:ENABle?')
    
    def set_display_mode(self, mode):
        # Set the display mode to V-V, I-I, or V-I
        if mode.upper() in ['V-V', 'I-I', 'V-I']:
            self.instr.write(':DISPlay:MODE ' + mode.upper())
        else:
            raise ValueError("Invalid display mode. Choose 'V-V', 'I-I', or 'V-I'.")

    # Additional methods can be added here for other SCPI commands
```
